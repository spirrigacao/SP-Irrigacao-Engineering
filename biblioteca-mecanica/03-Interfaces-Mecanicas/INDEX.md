# Interfaces Mecânicas

Esta pasta reúne todas as interfaces mecânicas utilizadas pelos produtos da SP Irrigação.

Uma interface mecânica representa a forma como dois componentes são acoplados mecanicamente.

Interfaces mecânicas não são produtos.

Elas representam geometrias padronizadas utilizadas pelos produtos da SP Irrigação.

Cada interface possui sua própria documentação técnica e pode ser utilizada por diversos produtos.
